# CADP End-to-End Repository

This repository contains full documentation, templates, and guidance
for creating **Consumer Aligned Data Products (CADPs)** on the NEO platform.

## Contents
- docs/ : Architecture, process, tools, modelling
- examples/ : Example DBT models
- checklists/ : Practical checklists
- templates/ : Starter YAML templates

This repo is documentation-first and can be safely shared.
