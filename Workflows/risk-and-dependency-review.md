# Risk and Dependency Review Workflow

## Trigger

- Review risks and dependencies
- Create risk register

## Required Input

- active initiatives
- dependencies
- blockers
- external constraints

## Steps

1. List top risks by impact and probability.
2. Map dependencies and critical path constraints.
3. Assign mitigation owner for each top risk.
4. Define escalation deadlines by severity.
5. Capture management attention items.

## Output

- risk register
- dependency map
- mitigation actions
- owners
- escalation deadlines

## Save Output

- Artifacts/risk-registers/YYYY-MM-DD-risk-review.md
- Use Templates/risk-register.md
