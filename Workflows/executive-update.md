# Executive Update Workflow

## Trigger

- Create executive update
- Summarize for management

## Required Input

- current status
- progress
- blockers
- risks
- decisions needed

## Steps

1. Produce a five-line summary first.
2. Report RAG status with evidence.
3. Highlight progress tied to outcomes.
4. Separate risks from noise and include mitigations.
5. State management asks and by-when.

## Output

- five-line summary
- RAG status
- key progress
- risks and mitigations
- management asks
- next steps

## Save Output

- Artifacts/executive-updates/YYYY-MM-DD-exec-update.md
- Use Templates/executive-summary.md
