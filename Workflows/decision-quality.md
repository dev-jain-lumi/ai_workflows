# Decision Quality Workflow

## Trigger

- Run decision quality
- Help me decide

## Required Input

- decision
- options
- constraints
- stakeholders
- deadline

## Steps

1. State the exact decision in one sentence.
2. Define at least 2 realistic options.
3. Compare tradeoffs: value, risk, cost, speed, reversibility.
4. Identify assumptions and confidence level.
5. Name decision owner and approver.
6. Define follow-up checkpoint and success criteria.

## Output

- recommendation
- tradeoffs
- reversibility
- decision owner
- assumptions
- risks
- follow-up checkpoint

## Save Output

- Artifacts/decision-logs/YYYY-MM-DD-decision-topic.md
- Use Templates/decision-record.md
